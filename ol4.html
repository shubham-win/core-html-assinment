HTML Tags

  1. <a>

    The anchor tag is used to create a hyperlink to another webpage or email address.

  2. <b>

    The bold tag is used to display text in a bold font.

  3. <body>

    The body tag is used to define the content of an HTML document.

  4. <br>

    The break tag is used to insert a line break in an HTML document.

  5. <div>

    The div tag is used to group elements together for styling or layout purposes.

  6. <h1>

    The h1 tag is used to define a heading element, with the number indicating the level of importance (h1 being the most important).

  7. <html>

    The html tag is used to define the root element of an HTML document.

  8. <i>

    The italic tag is used to display text in an italic font.

  9. <img>

    The img tag is used to embed an image in an HTML document.

  10. <p>

    The paragraph tag is used to define a paragraph of text.

  11. <span>

    The span tag is used to group inline elements together for styling or layout purposes.

  12. <strong>

    The strong tag is used to indicate strong importance or emphasis on a piece of text.

  13. <title>

    The title tag is used to define the title of an HTML document, which appears in the browser's title bar and in search engine results.
